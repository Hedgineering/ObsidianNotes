You are given an integer $n$ denoting the number of cities in a country, numbered:

$$0, 1, 2, \ldots, n-1$$

You are also given a 2D integer array $\text{roads}$, where each entry:

$$\text{roads}[i] = [a_i, b_i]$$

denotes a **bidirectional** road connecting cities $a_i$ and $b_i$.

## Goal

Assign each city a distinct integer value from the set:

$${1, 2, 3, \ldots, n}$$

such that each value is used **exactly once**. Define the **importance** of a road $(a_i, b_i)$ as:

$$\text{importance}(a_i, b_i) = \text{value}(a_i) + \text{value}(b_i)$$

Return the **maximum total importance** over all roads:

$$\text{Total Importance} = \sum_{i} \big(\text{value}(a_i) + \text{value}(b_i)\big)$$

---

## Constraints

$$2 \leq n \leq 5\times10^4$$ $$1 \leq \text{roads.length} \leq 5\times10^4$$ $$\text{roads}[i].\text{length} = 2$$ $$0 \leq a_i, b_i \leq n-1, \qquad a_i \neq b_i$$ $$\text{No duplicate roads}$$

---

## Key Insight (for solving)

Let $\deg(c)$ denote the **degree** of city $c$ — the number of roads incident to it:

$$\deg(c) = \big|{, i : a_i = c \text{ or } b_i = c ,}\big|$$

Then the total importance can be rewritten as a sum over cities rather than roads, since each city's value contributes once per incident road:

$$\text{Total Importance} = \sum_{c=0}^{n-1} \deg(c) \cdot \text{value}(c)$$

To **maximize** this weighted sum, by the rearrangement inequality, assign the **largest values to the cities with the highest degree**:

$$\text{value}(c) \propto \text{rank of } \deg(c) \text{ in descending order}$$

Concretely: sort cities by $\deg(c)$ descending, then assign values $n, n-1, \ldots, 1$ in that order.

**Complexity:** $O(n \log n)$ for sorting, $O(n + |\text{roads}|)$ for computing degrees and the final sum.

```py
def maximumImportance(self, n: int, roads: List[List[int]]) -> int:
    degree = [0] * n

    # calculate in-degree per node
    for src, dst in roads:
        degree[src] += 1
        degree[dst] += 1

    # sort by in-degree ascending
    assignments = [(d, i) for i, d in enumerate(degree)]
    assignments.sort()
  
    # assign importance based on index of sorted degree 
    # (higher importance for higher degree)
    node_importance = [0] * n
    for idx, pair in enumerate(assignments):
        degree, node = pair
        node_importance[node] = idx+1

    # sum importance per degree for road pairs
    res = 0
    for src, dst in roads:
        res += node_importance[src] + node_importance[dst]

    return res
```
