You are given a positive integer `num` consisting only of digits `6` and `9`.

Return _the maximum number you can get by changing **at most** one digit (_`6` _becomes_ `9`_, and_ `9` _becomes_ `6`_)_.

**Example 1:**

**Input:** num = 9669
**Output:** 9969
**Explanation:** 
Changing the first digit results in 6669.
Changing the second digit results in 9969.
Changing the third digit results in 9699.
Changing the fourth digit results in 9666.
The maximum number is 9969.

**Example 2:**

**Input:** num = 9996
**Output:** 9999
**Explanation:** Changing the last digit 6 to 9 results in the maximum number.

**Example 3:**

**Input:** num = 9999
**Output:** 9999
**Explanation:** It is better not to apply any change.

**Constraints:**

- `1 <= num <= 104`
- `num` consists of only `6` and `9` digits.

---
## Solution

We're only allowed to make one change, and to get a bigger number it makes sense to change the first $6$ we see into a $9$ otherwise leave the number alone if there are no $6$'s. 

```py
def maximum69Number (self, num: int) -> int:
    s = str(num) # can easily iterate through digits of a string
    i = -1

    for j in range(len(s)):
        if s[j] == '6': # find index of first '6'
            i = j
            break
            
	# if no '6's found, just return else replace pos i with a '9'
    return num if i == -1 else int(s[:i] + '9' + s[i+1:])
```
