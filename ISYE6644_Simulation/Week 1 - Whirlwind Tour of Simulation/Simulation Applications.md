# When should simulation be used?
- Stochastic (probabilistic) processes - scenarios or items that involve some random error, noise term, or non-deterministic component
- When there is no obvious closed form equation that solves the system
- When there is a lot of data to crunch or huge problem/solution space

## Specific Example Applications
- Manufacturing
	- \[Automobile, machine, carpet, textile, etc.] production facility
- Queueing Problems
	- Call center analysis (how many Representatives do we need to have x service time on average?)
	- Fast food drive thru
	- Airport Security Line
	- Generic supply lines
	- Financial Analysis
		- Options/ Stocks
		- Portfolio Optimization
